## Current Projects

### TOON vs JSON Comparison
Demonstrates the TOON format - a compact serialization format that reduces token usage by 20-60% compared to JSON, optimized for LLM applications.

**Run the demo:**
```bash
pip install -r requirements.txt
python toonVsJson.py
```
